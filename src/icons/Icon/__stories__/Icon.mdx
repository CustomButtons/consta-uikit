
# Icon

Иконка нужна там, где можно обойтись без слов.

## Свойства

<!-- props:start -->

| Свойство         | Тип                                     | По умолчанию  | Описание                                     |
| ---------------- | --------------------------------------- | ------------- | -------------------------------------------- |
| `view` | `'alert', 'brand', 'ghost', 'link', 'primary', 'secondary', 'success', 'warning'`| `primary`| Тип иконки |
| `size` | `'xs', 's', 'm'` | `'m'` | Размер иконки  |
| `className `      | `string`  | -  | Дополнительный CSS-класс  |

<!-- props:end -->
