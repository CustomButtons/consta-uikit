# Sidebar

Sidebar - компонент контейнер для контента и кнопок, с белым фоном и полупрозрачным оверлеем(опционально), расположенный по стороне на экране.

## Список свойств

| Свойство         | Тип или варианты значения          | По умолчанию           | Описание                                          |
| ---------------- | ---------------------------------- | ---------------------- | ------------------------------------------------- |
| isOpen?          | `boolean`                          | -                      | флаг "открыто"                                    |
| onClose?         | `() => void`                       | -                      | событие которое отработает по закрытию            |
| onOpen?          | `() => void`                       | -                      | событие которое отработает по открытию            |
| hasOverlay?      | `boolean`                          | -                      | отображать `Overlay` ?                            |
| onOverlayClick?  | `(e: React.MouseEvent) => void`    | -                      | событие которое отработает при клике на `Overlay` |
| position?        | `'right', 'bottom', 'left', 'top'` | `right`                | позиция                                           |
| width?           | `'auto'`                           | `auto`                 | ширина                                            |
| height?          | `'auto'`                           | `auto`                 | высота                                            |
| className?       | `string`                           | -                      | дополнительный ксласс компонента                  |
| windowClassName? | `string`                           | -                      | дополнительный ксласс для элемента `Window`       |
| children?        | `React.ReactNode`                  | -                      | контент                                           |
| container?       | `HTMLDivElement, undefined`        | `window.document.body` | елемент в котором рендериться компонент           |

## Пример использования

```ts
<Sidebar
  className={cnSidebarStories('Sidebar')}
  isOpen={isSidebarOpen}
  onClose={() => console.log('Коллбэк на закрытие')}
  onOpen={() => console.log('Коллбэк на открытие')}
  hasOverlay={hasOverlay}
  onOverlayClick={() => setIsSidebarOpen(false)}
  width={width}
  height={height}
  position={position}
>
  <Sidebar.Content className={cnSidebarStories('Content')}>
    <Text as="p" size="l" view="primary" weight="semibold" className={cnSidebarStories('Title')}>
      Заголовок сайдбара
    </Text>
    <Text as="p" size="m" view="secondary" className={cnSidebarStories('Body')}>
      Sed porta sollicitudin purus sagittis elementum. Pellentesque habitant morbi tristique
      senectus et netus et malesuada fames ac turpis egestas. Pellentesque tincidunt, augue et
      placerat cursus, neque sem imperdiet mi, ut fringilla nunc nibh non enim. Vivamus id metus
      convallis, commodo magna vel, scelerisque nisi.
    </Text>
  </Sidebar.Content>
  <Sidebar.Actions className={cnSidebarStories('Actions')}>
    <Button
      size="m"
      view="clear"
      label="Ок"
      width="default"
      onClick={() => setIsSidebarOpen(false)}
    />
    <Button
      size="m"
      view="ghost"
      label="Закрыть"
      width="default"
      onClick={() => setIsSidebarOpen(false)}
    />
  </Sidebar.Actions>
</Sidebar>
```
